Introduction

Welcome! On this website, you will learn how to create your first basic website.

Your first HTML file

  1. Create a new file: Open a text editor (like Notepad or Visual Studio Code) and create a new file. Save it with the .html extension (e.g.,index.html).

  2. Start with the HTML structure: Your document should have a basic structure, which includes a <!DOCTYPE> declaration, an opening <html> tag, a <head> section, and a <body> section. Here is an example: 


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" 
    content="width=device-width, 
    initial-scale=1.0">
    <title>My First HTML Page</title>
</head>
<body>
    <h1 class="yellowfy">Hello, World!</h1>
    <p>This is my first HTML document.</p>
</body>
</html>

Linking a css sheet

  1. Create a new file: Open a text editor (like Notepad, Visual Studio Code, or Sublime Text) and create a new file. Save it with a .css extension, for example, styles.css.

  2. Link your CSS file to your HTML: In your HTML file, link the CSS file within the <head> section using the <link> tag: 


<link rel="stylesheet" 
type="text/css" href="styles.css">

Write your CSS rules: In your styles.css file, use CSS selectors to target HTML elements and define their styles. For example:


/*This affects the paragraph tag.*/
p{
    color: blue;  /* Sets text color to blue */
    font-size: 16px; /* Sets font size to 16px */
}

/* Sets background color to yellow 
in this instance we use this class to set color 
of h1 to yellow*/
.yellowfy {
    color: yellow; 
}
